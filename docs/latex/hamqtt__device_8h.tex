\doxysubsection{Detailed Description}
Public API for representing a Home Assistant MQTT device. 

This header provides the {\ttfamily \doxylink{struct_h_a_m_q_t_t___device}{HAMQTT\+\_\+\+Device}} type and functions for managing a device with one or more MQTT components in Home Assistant. It allows registration of components, setting device metadata, and updating MQTT state.

Components are dispatched using the \doxylink{struct_h_a_m_q_t_t___component}{HAMQTT\+\_\+\+Component} interface.
