\doxysubsection{Detailed Description}
Interface definition for Home Assistant MQTT components. 

This header defines the abstract {\ttfamily \doxylink{struct_h_a_m_q_t_t___component}{HAMQTT\+\_\+\+Component}} interface and the virtual dispatch function declarations for MQTT-\/based Home Assistant components. Components implementing this interface must provide their own VTable implementation for MQTT discovery, messaging, and updates.
