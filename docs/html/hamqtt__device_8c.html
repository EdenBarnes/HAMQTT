Detailed Description

Implementation of the HAMQTT_Device MQTT device abstraction.

This file defines the internal structure and logic of the HAMQTT_Device, including device metadata handling, MQTT discovery publishing, message routing, and lifecycle management of components.

This is the core implementation used by device applications integrating with Home Assistant's MQTT discovery protocol.
